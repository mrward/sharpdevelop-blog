To use a type in your application you need to add a reference to the assembly where that type is defined. A project created from a SharpDevelop project template will typically have a few default references already added to the project, for example, the VB.NET and C# Windows Application templates have the following references already added

  • System
  • System.Data
  • System.Drawing
  • System.Windows.Forms
  • System.Xml

Adding a Reference

  1. From the View menu select Projects .

    This opens the Project Explorer.

  2. Select the name of the project in the Project Explorer , right click and select Add Reference .

    Alternatively you can select Add Reference from the Project menu.

    This opens the Add Reference dialog box.

    From the dialog you can add a reference to an assembly in the GAC, a project reference, a COM reference or you can add a reference by browsing to an assembly on the file system.

  3. Select the desired assembly and click the Select button. Multiple assemblies can be selected by holding down the Ctrl key. Selected references are displayed in the Selected References list at the bottom of the dialog. The screenshot below shows the System.Runtime.Remoting and the System.Web assemblies selected and ready to be added to the project.

  4. Add the reference to your project by clicking the OK button.

    The reference will be displayed under References in the Project Explorer. The screenshot below shows the System.Runtime.Remoting and System.Web references added to the project.

Removing a Reference

  1. Open the Project Explorer.
  2. Expand the References by clicking the plus icon .
  3. Select the reference you want to remove, right click and select Remove .

    The reference will then be removed from the project.
