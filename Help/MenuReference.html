File Menu

Menu Item Shortcut Description
New, File... Ctrl+N Opens the new file dialog box. If a project is open you will be asked whether the file should be added to the currently selected project. If no project is currently open a new file is opened inside SharpDevelop.
New, Solution... Ctrl+Shift+N Opens the New Project dialog box. After choosing a project template and clicking the OK button a new solution containing this new project will be opened in SharpDevelop.
Open, File... Ctrl+O Opens the Open File dialog box allowing you to browse for a file and open it inside SharpDevelop.
Open, Project/Solution... Ctrl+Shift+O Opens the Open File dialog box allowing you to browse for a project (e.g. .csproj, .vbproj) or a solution (.sln) file and open it inside SharpDevelop.
Close, File Ctrl+F4 Closes the file currently being viewed.
Close, Solution Ctrl+Shift+F4 Closes the currently open solution.
Reload file Ctrl+Shift+U Reloads the file currently being viewed from disk.
Save Ctrl+S Saves any changes the file currently being viewed.
Save As... Opens the Save As dialog box allowing you to save a copy of the file being viewed to another location.
Print Ctrl+P Opens the Print dialog allowing you to print the file currently being viewed.
Print Preview Opens the Print Preview dialog showing the file currently being viewed.
Recent Files Contains a list of the most recently opened files. Selecting a filename from the list opens it inside SharpDevelop.
Recent Files, Clear recent files list Clears the list of most recently opened files.
Recent Projects Contains a list of the most recently opened projects. Selecting a filename from the list opens the project inside SharpDevelop.
Recent Projects, Clear recent projects list Clears the list of most recently opened projects.
Exit Closes SharpDevelop. Before closing SharpDevelop will prompt the user to save any unsaved changes made to the project and currently open files.

Edit Menu

Menu Item Shortcut Description
Undo Ctrl+V Causes the last edit action to be undone.
Redo Ctrl+Y Causes the last edit action that was undone to be redone.
Cut Ctrl+X Removes the item and copies it into the clipboard.
Copy Ctrl+C Copies the selected item into the clipboard.
Paste Ctrl+V Pastes the item from the clipboard.
Delete Deletes the selected item.
Format, Remove Leading Ws Removes all the whitespace at the start of the line in the file being edited.
Format, Remove Trailing Ws Removes all the whitespace at the end of the line in the file being edited.
Format, To Upper Case Upper cases the selected text.
Format, To Lower Case Lower cases the selected text.
Format, Capitalize This capitalizes the first letter of all the words found in the selected text.
Format, Invert Case Inverts the case of the characters in the selected text.
Format, Sort Lines... Opens the Sort Lines Options dialog box.
Format, Tabs to Spaces Converts all tabs to spaces in the currently selected text. One tab will be converted into N spaces, where N is the currently defined tab indentation size.
Format, Spaces to Tabs Converts all spaces to tabs in the currently selected text. N spaces will be converted into one tab, where N is the currently defined tab indentation size.
Format, Leading Tabs to Spaces Converts tabs at the beginning of the line to spaces for the selected lines. One tab will be converted into N spaces, where N is the currently defined tab indentation size.
Format, Leading Spaces to Tabs Converts spaces at the beginning of the line to tabs for the selected lines. N spaces will be converted into one tab, where N is the currently defined tab indentation size.
Format, Comment Region Comments the selected lines. For example, with a C# file this adds // to the start of all the selected lines.
Format, Indent Ctrl+I Intelligently indents the selected text based on the type of document open.
Folding, Toggle fold Ctrl+Shift+M Expands or collapses the fold at the current cursor position.
Folding, Toggle all folds Ctrl+Shift+L Expands or collapses the fold at the current cursor position and all folds contained inside the parent fold.
Folding, Show definitions only Ctrl+Shift+P Folds code lines so only the class and method definitions are shown, all the code inside a method is hidden inside the fold.
Select All Ctrl+A Selects all the text in the currently open file.
Word Count... Opens the Word Count dialog box.

Project Menu

This menu is only shown when a project has been opened inside SharpDevelop and the project is selected in the Projects Explorer.

Menu Item Shortcut Description
Add, New Item... Opens the New File dialog box. The created file will be added to the project.
Add, Existing Item. Shows the Open File dialog box allowing you to add existing files to the project.
Add, New Folder Adds a new folder to the project.
Add Reference Opens the Add Reference dialog box.
Add Mono Reference Opens the Add Mono Reference dialog box. This menu item is only available if Mono is installed.
Add Web Reference Opens the Add Web Reference dialog box.
Run Project Runs the project under the debugger.
Set as StartUp Project In a solution that contains multiple projects the currently selected project will be the one that is run.
Html Export Opens the Export to Html dialog box. This creates html web pages for the project's source code files.
Convert Converts the project from one language to another. SharpDevelop can convert in both directions between the Boo, C# and VB.NET languages.
Build documentation with NDoc Runs NDoc.
NAnt, Run Default Target Runs NAnt and builds the default target for the NAnt build file in the current project.
NAnt, Run Clean Target Runs the clean target for the NAnt build file in the current project.
NAnt, Stop NAnt Terminates the currently executing NAnt build.
Project Options... Opens the Projects Options.

Build Menu

Menu Item Shortcut Description
Build Solution F8 Builds the currently open solution. This runs the default target in the MSBuild project file.
Rebuild Solution Alt+F8 Cleans and then builds the currently open solution. This runs the Rebuild target in the MSBuild project file.
Clean Solution Cleans the solution. This runs the Clean target in the MSBuild file.
Build [Project Name] F9 Builds the selected project.
Rebuild [Project Name] Alt+F9 Rebuilds the selected project.
Clean [Project Name] Cleans the selected project.
Set configuration Shows a list of configurations that the solution contains with the currently active configuration checked.

Debug Menu

Menu Item Shortcut Description
Run F5 Builds the project and then runs the built application under the debugger.
Run without debugger Ctrl+F5 Builds and runs the application without attaching the debugger to it.
Stop process Terminates the application currently being debugged.
Break Ctrl+Alt+B Causes the debugger to stop and break into the executing application.
Continue debugging F6 Continues the execution of the application after it has been paused by the debugger after hitting a breakpoint.
Step over F10 Executes the current source code line, but does not enter the method.
Step into F11 Enters the method, if any, on the current source code line and stops on the first line of the method. If there is no method then the current line is executed.
Step out Shift+F11 Completes execution of the current method and returns to its parent method.
Toggle Breakpoint F7 Enables or disables a breakpoint at the currently selected source code line.

Search Menu

Menu Item Shortcut Description
Find... Ctrl+F Opens the Search dialog box.
Find Next F3 Finds the next text that matches the current search criteria.
Replace... Ctrl+R Opens the Replace dialog box.
Toggle Bookmark Ctrl+F2 Enables or disables the bookmark at the currently selected source code line.
Prev Bookmark Alt+F2 Moves the cursor in the text editor to the previous bookmark in the currently viewed file.
Next Bookmark F2 Moves the cursor in the text editor to the next bookmark in the currently viewed file.
Clear All Bookmarks Clears all the bookmarks defined in the currently viewed file.
Goto Line Number... Ctrl+G Opens the Goto Line dialog box.
Goto Matching Brace Ctrl+B Moves the cursor in the text editor to the brace that matches the brace at the current cursor position.

XML Menu

This menu is only available when an XML file is currently being viewed.

Menu Item Shortcut Description
Create Schema Infers a schema based on the currently viewed XML file and opens a new file with the inferred schema.
Validate XML Ctrl+Shift+V Validates the viewed XML file against its schema.
Assign XSLT... Associates an XSL transform file with the XML file.
Open XSLT Opens the XSL transform that is associated with the viewed XML file.
Run XSLT Ctrl+Shift+T Runs the XSL transform on the XML file.

Window Menu

Menu Item Shortcut Description
Next Window Ctrl+Tab Moves from left to right to the next open file window.
Prev Window Ctrl+Shift+Tab Moves back from right to left to the previous file window.
Split Splits the text editor view in two halves. Each half can be separately scrolled and edited.
Close All Documents Closes all opened files.
