Thank you for choosing SharpDevelop as your development environment for .NET. SharpDevelop is a state of the art development environment, written from scratch using nothing but .NET itself - even the C# source code is available to you, allowing you to see the possibilities offered by .NET.

Features at a glance:

  • Code completion
  • Code auto-generation
  • C#, VB.NET, Boo, Gtk#, Glade#, console, Windows Forms project and file templates.
  • Forms Designer for C#, VB.NET and Boo.
  • Integrated debugger
  • Code coverage
  • Integrated NAnt support
  • XML editor
  • Unit testing
  • Web references
  • PInvoke signatures
  • Class browser
  • Code refactoring
  • XML documentation preview
  • Language conversion (C# to VB.NET, C# to Boo, and vice versa)
  • Bookmarks
  • Project browser
  • File browser
  • Regular expressions toolkit
  • Targeting the .NET Frameworks 1.1, 1.0
  • Supports the Mono Framework 1.1, 2.0 and Mono C# compilers
  • Uses the same project and solution file format as Microsoft's Visual Studio 2005 and Visual Studio Express.
  • Extensible through plug-ins.

Please note that SharpDevelop is a work in progress. Visit our Web site regularly to learn about updates, suggest features, contribute code and inform us about problems you have encountered.

Thank you, the SharpDevelop Team.
