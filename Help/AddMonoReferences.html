If you have Mono installed you can add references to items in Mono's GAC using the Add Mono Reference dialog box.

Adding a Mono Reference

  1. From the View menu select Projects .

    This opens the Project Explorer.

  2. Select the name of the project in the Project Explorer , right click and select Add Mono Reference .

    Alternatively you can select Add Mono Reference from the Project menu.

    This opens the Add Mono Reference dialog box.

    From the dialog box you can add a reference to an assembly in Mono's GAC, a project reference, or you can add a reference by browsing to an assembly on the file system.

  3. Select the desired assembly and click the Select button.
  4. Add the reference to your project by clicking the OK button. The reference will be displayed under References in the Project Explorer .
