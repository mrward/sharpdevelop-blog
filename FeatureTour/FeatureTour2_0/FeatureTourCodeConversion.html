A lot of code examples are only available in one language, usually C#, and often you need the code in another language. SharpDevelop can convert the code between the three languages C#, VB.NET and Boo. It can convert individual files or an entire project to another language.

To convert an open project, from the Project menu select Convert and choose the desired language.

To convert a single file, from the Tools menu select one of the Convert menu items.
